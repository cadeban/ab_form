# Rad Band Form
## About

Purchase Tickets form for rad touring bands.

## Installation
Run in terminal:
1. `npm i`
2. `npm run dev`
3. Navigate to `http://localhost:5173/` in browser
4. Consider purchasing some rad band tickets
